# System packages required: ffmpeg, tesseract-ocr
#
# TTS engine: Chatterbox (Resemble AI) is still installed SEPARATELY in prod
# (`pip install chatterbox-tts`) — it isn't on PyPI-friendly terms for CPU dev
# machines. The API boots fine without it; only actual TTS calls require it. With
# the pins below, that install adds only the chatterbox package + its light deps.
# ─────────────────────────────────────────────────────────────────────────────
