# ─────────────────────────────────────────────────────────────────────────────
# Phansora API — consolidated dependencies (spokenverse + chrono_origin + dossier_nova)
#
# Torch baseline: aligned to the IndexTTS2 TTS engine (torch 2.8.0, CPU-only build).
# torchvision 0.23.0 / torchaudio 2.8.0 are the matching releases. We pin the `+cpu`
# wheels deliberately: the prod GPU (~4 GB) is too small to load IndexTTS2, so TTS runs
# on CPU. CPU wheels also avoid pulling the multi-GB nvidia-cuda-* packages, and make
# torch.cuda.is_available() False so IndexTTS2 auto-selects CPU. For a CUDA host with
# enough VRAM (~8 GB+), swap `whl/cpu` + `+cpu` for a `whl/cu126` + `+cu126` build.
#
# System packages required: ffmpeg, tesseract-ocr
#
# TTS engine: IndexTTS2 is a git checkout (NOT a published pip package), run in-process.
# In prod: run `make install` (installs the torch/transformers pins below), then clone it
# and `pip install -e /var/www/index-tts` INTO this venv, install pynini via conda
# (`conda install -c conda-forge pynini==2.1.6`), download the IndexTeam/IndexTTS-2
# checkpoints, and set INDEXTTS2_REPO. The API boots without it; only TTS calls require
# it.  ⚠ IndexTTS2 pins `torch==2.8.*` / `transformers==4.52.1`, both satisfied by the
# pins below — so install IndexTTS2 AFTER make install and pip leaves torch untouched.
# ─────────────────────────────────────────────────────────────────────────────

# Torch wheels are pinned as DIRECT CloudFront URLs (download.pytorch.org) rather than
# via `--extra-index-url .../whl/cpu`. The index now hands out download-r2.pytorch.org
# (Cloudflare R2) links, and the prod network cannot complete a TLS handshake to
# Cloudflare (SSLV3_ALERT_HANDSHAKE_FAILURE / alert 40, IPv4 + IPv6). The
# download.pytorch.org path serves the same wheels from Amazon CloudFront, which is
# reachable. These URLs are pinned to cp310 / manylinux_2_28_x86_64 — the prod image
# (torch >= 2.7 uses the manylinux_2_28 tag, NOT the old linux_x86_64). For a CUDA host,
# swap `whl/cpu` + `+cpu` for `whl/cu126` + `+cu126`.
torch @ https://download.pytorch.org/whl/cpu/torch-2.8.0%2Bcpu-cp310-cp310-manylinux_2_28_x86_64.whl
torchvision @ https://download.pytorch.org/whl/cpu/torchvision-0.23.0%2Bcpu-cp310-cp310-manylinux_2_28_x86_64.whl
torchaudio @ https://download.pytorch.org/whl/cpu/torchaudio-2.8.0%2Bcpu-cp310-cp310-manylinux_2_28_x86_64.whl
